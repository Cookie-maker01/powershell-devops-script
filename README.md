# 🚀 PowerShell DevOps Automation Script

This project provides a reusable PowerShell script to automate development workflows.

## ✨ Features

* Detects project type (Node.js / .NET)
* Automatically installs dependencies
* Pulls latest Git code
* Starts development server
* Supports environment parameters (dev / prod)

## ▶️ Usage

```powershell
.\start.ps1 -env dev
```
